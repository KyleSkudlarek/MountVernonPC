  • Faulty RAM is commonly responsible for a variety of computer problems such as random shutdowns, BSOD's and sluggish performance.
  • RAM is the computer's "working memory." It allows the computer to handle more processes running at any given time and gives it the resources to execute complicated, memory-costly programs effectively.
  • Hard drives (HDD's) are another usual suspect for computer malfunctioning. This is where the computer's "long term memory" is held such as pictures, files, downloads, etc. If you need more "space" on your computer you can either delete old files or install an additional hard drive.
  • I handle any sized problem relating to your computer's components, from swapping out RAM to building you an entirely new computer.
  • A minimum of 4 GB's of RAM is recommended for running today's OS's optimally and performing basic tasks like web browsing and file editing. If you notice sluggish performance watching HD video's on Netflix, running multiple programs at once, playing games, etc. adding an additional GB or two of RAM could make all the difference for minimal cost relative to buying a whole new pc.
